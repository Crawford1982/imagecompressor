## Private Image Compressor

Private Image Compressor is a free, privacy-first image compression tool that runs entirely in your browser. Images are never uploaded to a server – everything happens locally on your device.

- **Live site**: https://privateimagecompressor.com/
- **Key features**:
  - Client-side compression (no uploads, no storage)
  - Supports JPG, PNG, and WebP
  - Adjustable quality slider with live size and % saved
  - Sensible size limits (20 MB on phones, up to 50 MB on desktop)
  - GDPR-friendly analytics with consent banner

The repository is designed to be hosted as a static site (for example, via GitHub Pages) and requires no backend services or build step.
